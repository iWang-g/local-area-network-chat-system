# UI 参考图索引

本文件记录**仿 QQ 风格**界面实现时所对照的参考截图，便于设计与答辩时说明「布局与交互借鉴常见 PC 即时通讯产品，美术资源为原创或授权素材」。

## 图片目录

所有参考 **PNG** 放在仓库根目录下的专用文件夹（与 `docs/` 同级）：

**`ui参考图/`**

后续新增图片时：将文件放入该目录，并在下表**按文件名追加一行**说明用途。

## 索引表

每在 `ui参考图/` 中增加或重命名一个 PNG，在下方表格**追加一行**（勿只改文件名不更新表）。

| 文件名 | 简要说明 | 对应界面 / 开发阶段 |
|--------|----------|---------------------|
| `登录界面.png` | PC 端 IM 登录页布局与配色参考 | `LoginWidget`（`qt-client/src/ui/login_widget.cpp`） |
| `聊天主界面.png` | 主窗三栏 / 会话 + 聊天区参考 | 后续主界面阶段 |

## 在 Markdown 中引用（可选）

在其它 `docs` 文档里如需插入预览，可使用相对路径，例如：

```markdown
![登录参考](../ui参考图/你的文件名.png)
```

在 PDF/Word 中排版时，可直接从 `ui参考图` 文件夹插入对应 PNG。

## 维护约定

1. **命名**：建议 `模块-场景-序号.png`（如 `friend-list-01.png`），避免中文文件名在部分工具链下异常；若已使用中文文件名也可保留，但表中登记**准确文件名**。  
2. **版权**：参考图仅作布局与交互对照，**勿**将受版权保护的界面截图直接用作软件内资源；程序内图标、背景等使用自绘、开源或授权素材。  
3. **同步**：每增加或删除 `ui参考图` 下的文件，**同步更新本索引表**，避免文档与目录不一致。
